## 🚀 DEPLOYMENT IN 3 SIMPLE STEPS

### STEP 1: Upload to GitHub (5 minutes)
```powershell
cd "c:\Users\dell\Desktop\MY WORK\school-bloom-desk-main"
git init
git add .
git commit -m "School Management System Ready"
# Create repo at https://github.com/new
# Copy lines and run them
```

### STEP 2: Deploy Frontend to Vercel (3 minutes)
1. Go to https://vercel.com/import
2. Select your GitHub repo
3. Click Deploy
4. **DONE!** Get your frontend URL

### STEP 3: Deploy Backend to Railway (5 minutes)
```powershell
npm install -g @railway/cli
railway login
cd backend
railway up
```
4. **DONE!** Get your backend URL

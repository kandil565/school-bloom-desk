# School Operations Dashboard (SIOMS)

Comprehensive school operations and HR management system.

## Project info

This project manages various school aspects such as Staff Attendance, HR Management, Payroll, Canteen, Inventory, Suppliers, Workshops, Assets Custody, and Reports.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
